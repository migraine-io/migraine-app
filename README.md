# Usage

### 1. Install Redis

```bash
$ brew install redis
```

Follow the instructions on screen to start the serve.

### 2. Install the gems

```bash
$ gem install bundler
$ bundle install

```

### 3. Start the server

```bash
$ make server
```

### 4. Done!

Go to [http://localhost:9393](http://localhost:9393)


**Requires Ruby 2.1+**
