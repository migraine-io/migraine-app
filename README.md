# Usage

### 1. Install Redis

```bash
$ brew install redis
```

Follow the instructions on screen to start the serve.

### 2. Install the gems

```bash
$ gem install cuba
$ gem install mote
$ gem install mote-render
$ gem install ohm
$ gem install scrivener
```

### 3. Start the server

```bash
$ rackup config.ru
```

### 4. Done!

Go to [http://localhost:9292](http://localhost:9292)


**Requires Ruby 2.1+**
